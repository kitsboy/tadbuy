# ZapCampaign - Android App with Real Nostr Zaps & Bitcoin/Lightning

**FOSS Android Studio app** (Kotlin) for managing campaigns funded by **Nostr Zaps** (NIP-57). 

## Features (no UI design changes - tile-based dashboard preserved)
- **Nostr Tile**: Connects to real relay (`wss://relay.damus.io` and `wss://relay.primal.net`). Publishes events and handles Zaps.
- **Lightning/Bitcoin Tile**: LNURL-pay and sats via Zaps (uses existing TadBuy Lightning service).
- **Campaigns Tile**: Loads data from Agent API.
- **Agent API Integration**: New endpoint in `../server.ts` at `/api/agent/campaigns` for Nostr agents/bots to query campaigns (JSON with npub, relay, goalSats).

1. Open in **Android Studio** (2024.3+ recommended, see https://developer.android.com/studio/intro).
2. The project uses:
   - Min SDK 24, Target SDK 35
   - OkHttp for WebSocket to Nostr relay
   - Kotlinx serialization for JSON
   - Tile layout (4 cards: Nostr, Lightning, Bitcoin, Campaigns) - **design unchanged**
3. Add NostrService.kt for relay connection and Zap publishing (basic NIP-57 event creation).
